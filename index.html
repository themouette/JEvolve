Example

Get a look at phpdocumentor generated doc

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/themouette/JEvolve

Install

If you installed phpdocumentor through pear, then just run the deploy.sh script: 

$ ./deploy.sh
you might be asked for root access as the pear folders are owend by root.

To deploy the theme in a custom phpdoc installation, do the following (PDOC_PATH beeing your installation path) 

$ mkdir PDOC_PATH/Converters/HTML/Smarty/JEvolve
$ cp -R GIT_REPO/* PDOC_PATH/Converters/HTML/Smarty/JEvolve/

to use it without modifying any phpdoc installation, you have to clone the repo using following: 

$ git clone git://github.com/themouette/JEvolve Converters/HTML/Smarty/JEvolve
you now can now use templatebase option.

Usage

If you installed the theme in phpdocumentor directory, simply use 

phpdoc -d './phpdocumentor' -t '/var/www/html/docs/' -o HTML:Smarty/JEvolve:default -s on
If you installed it to take advantage of templatebase option: 
phpdoc -d './phpdocumentor' -t '/var/www/html/docs/' -tb /PATH/TO/REPO -o HTML:Smarty/JEvolve:default -s on

If you like better a phpdoc.ini file: 

phpdoc -c phpdoc.ini
with following minimal phpdoc.ini: 
        #!ini
        directory   = ./phpdocumentor
        #ignore      = 
        target      = /var/www/html/docs/
        quiet       = true
        title       = Your Project
        sourcecode  = on
        #templatebase = /PATH/TO/REPO
        output      = HTML:Smarty/JEvolve:default
        readmeinstallchangelogSpecify = README
